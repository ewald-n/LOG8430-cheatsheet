# Architecture vs. Conception

## Tableau Comparatif

| Architecture | Conception |
| :--- | :--- |
| Haut niveau, squelette, déploiement | Bas niveau, code, classes, méthodes |
| Exigences **fonctionnelles + non-fonctionnelles** | Principalement **fonctionnelles** |
| Communication entre composants | Portée des classes, implémentation |

---

## Architecture
- **Niveau :** Haut niveau
- **Focus :** Squelette, déploiement
- **Exigences :** Fonctionnelles + non-fonctionnelles
- **Portée :** Communication entre composants

## Conception
- **Niveau :** Bas niveau
- **Focus :** Code, classes, méthodes
- **Exigences :** Principalement fonctionnelles
- **Portée :** Portée des classes, implémentation
