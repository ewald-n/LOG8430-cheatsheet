# LOG8430 - Architecture Logicielle & Mégadonnées

## Aide-Mémoire Examen Final

This repository contains a comprehensive cheat sheet for the LOG8430 final exam covering:

- **SOLID Principles**
- **Architectural Styles** (Distributed vs Non-distributed)
- **CAP Theorem** (Consistency, Availability, Partition Tolerance)
- **ACID vs BASE**
- **NoSQL Databases** (Key-Value, Document, Wide Column, Graph)
- **Blockchain** (PoW vs PoS, Blockchain Trilemma)
- **MapReduce & Streaming** (Hadoop, Spark, Kafka)
- **Message-Oriented Middleware** (RabbitMQ vs Kafka)
- **Event-Driven Architecture** (Reactor, Proactor, ACT, Acceptor-Connector)
- **SOA Communication** (SOAP, REST, RPC, GraphQL, gRPC)
- **AOP** (Aspect-Oriented Programming)
- **Microservices vs SOA**
- **Design Patterns & Antipatterns**

## Usage

Open `LOG8430_cheatsheet.html` in your web browser to view the formatted cheat sheet.

## Format

The cheat sheet is designed to be printed on 6 pages (A4 format) with optimized spacing and color-coding for different topics.
